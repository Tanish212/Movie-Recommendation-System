# MOVIE RECOMMENDATION SYSTEM


Built a content-based **Movie Recommendation System** using Python and Pandas for data preprocessing, TF-IDF for feature extraction, and Cosine Similarity to recommend movies with similar content. Enhanced user experience by integrating approximate movie title matching with difflib.

## FEATURES:

• Content-Based Recommendation using TF-IDF and Cosine Similarity.

• Movie Search by title.

• Approximate Title Matching using difflib to handle spelling mistakes.

• Data Preprocessing using Pandas (handling missing values and combining text features).

• Personalized Recommendations by finding movies with similar content.

• Fast Similarity Search using a precomputed cosine similarity matrix.

• Scalable Design that can work with large movie datasets.
